Installation steps

1. Create subdirectory in your HOME directory

2. Copy 'PlanetCNCL64_<version>.tar.gz' tarball to subdirectory

3. Extract 'PlanetCNCL64_<version>.tar.gz' tarball using: 
			tar -xvf PlanetCNCL64_*.tar.gz

4. Run installation script using:
			sh install.sh
			
5. You will need to type your 'sudo' password.
